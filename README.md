sharpPanel
==========

cPanel® API for .NET

#What is cPanel?
cPanel® is a control panel available on our Linux-based hosting accounts and servers. It lets you easily manage many aspects of your account.
You can read more at https://cpanel.net/cpanel-whm/

#What is sharpPanel?
sharpPanel is .NET library that lets you to use cPanel® API from .NET code. It is written in C# code.

#Notes 
Some comments in source code (i.e. methods parameters description) are taken from http://docs.cpanel.net/
